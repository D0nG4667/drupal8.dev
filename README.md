drupal8.dev
===========

Vagrant machine for drupal 8

1.- Clone this repo
```bash
$ git clone https://github.com/hechoendrupal/drupal8.dev.git
$ cd drupal8.dev
$ vagrant up

```

2.- Add VM to hosts file
```bash
  192.168.9.10 drupal8.dev

```

3.- Access your VM via drupal8.dev
